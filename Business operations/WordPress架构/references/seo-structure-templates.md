# SEO 页面结构模板

本文档提供不同页面类型的 SEO 结构模板，帮助快速搭建符合 SEO 最佳实践的页面。

---

## 1. 博客文章（How-to/Tutorial）

### 结构模板

```
H1: [如何] [做某事] - [年份] 指南

  H2: 为什么 [主题] 很重要
    - 统计数据
    - 痛点说明
    - 解决后的好处
  
  H2: [主要解决方案]（分步骤）
    H3: 步骤 1: [具体行动]
    H3: 步骤 2: [具体行动]
    H3: 步骤 3: [具体行动]
    ...
  
  H2: 常见错误与避坑指南
    - 错误 1
    - 错误 2
    - 错误 3
  
  H2: 工具/资源推荐
    - 工具 1
    - 工具 2
  
  H2: 常见问题 (FAQ)
    H3: 问题 1
    H3: 问题 2
    H3: 问题 3
  
  H2: 总结 + CTA
```

## 7. 关键 SEO 元素清单

每种页面类型必备：

### 基础 SEO
- [ ] H1 唯一且包含主关键词
- [ ] Title Tag 50-60 字符
- [ ] Meta Description 150-160 字符
- [ ] URL 简洁易读

### 内容结构
- [ ] H2 标题 2-5 个
- [ ] 至少 1 个 H2 包含关键词
- [ ] 内容长度适中（博客 >1000 字，产品页 >500 字）

### 内链
- [ ] 至少 2-3 条内链
- [ ] 链接到相关页面
- [ ] 使用描述性锚文本

### 用户体验
- [ ] 清晰的 CTA
- [ ] 移动端友好
- [ ] 图片优化（alt/尺寸）
- [ ] 快速加载（<3 秒）

### Schema Markup（根据页面类型）
- 博客：Article
- 产品：Product
- 本地业务：LocalBusiness
- FAQ：FAQPage
